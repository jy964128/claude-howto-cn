# 贡献者行为准则

## 我们的承诺

我们致力于为所有人提供一个热情友好、鼓舞人心的社区。我们承诺，每个人在我们的社区中参与都不会受到骚扰，无论年龄、体型、种姓、残疾、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何。

## 我们的标准

### 尊重他人
- 使用友好、包容的语言
- 尊重不同的意见、观点和经历
- 优雅地接受建设性批评
- 尊重文化和语言差异

### 协作精神
- 共同解决冲突
- 肯定他人的贡献
- 帮助他人学习和成长
- 分享知识，提出问题

### 专业态度
- 保持讨论聚焦和富有成效
- 避免贬低或侮辱性言论
- 在 issue 和讨论中保持主题
- 使用清晰、尊重的沟通方式

### 包容性
- 欢迎新贡献者
- 帮助新人上手项目
- 提供建设性反馈
- 包容不同观点的人

## 不可接受的行为

### 骚扰和歧视
- 关于性别、性别认同、性取向、残疾、精神疾病、外貌、体型、年龄、种族或宗教的攻击性言论
- 故意错误性别称呼或使用弃用姓名
- 无端或离题的性图像或行为

### 滥用行为
- 针对他人的暴力威胁或暴力语言
- 蓄意恐吓、跟踪或骚扰
- 发布他人隐私信息（人肉搜索）
- 恶意挑衅或蓄意干扰

## 适用范围

本行为准则适用于：
- 所有社区空间（GitHub issues、PR、讨论等）
- 项目仓库和文档
- 代表项目的公开活动
- 与项目事务相关的私人通信

## 执行

如果您经历或目击不可接受的行为，请通过以下方式报告：
1. **GitHub Issues**：私有报告功能
2. **直接消息**：直接联系项目维护者

**所有报告将保密处理。**

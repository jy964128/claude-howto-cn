# Claude How To - 设计系统

## 视觉识别

### 图标设计概念：带代码括号的指南针

Claude How To 图标使用**带有 `>` 代码括号的指南针**来代表代码世界的引导式导航：

```
     N（绿色）
     ▲
     │
W ───>─── E     指南针 = 引导/方向
     │          > 括号 = 代码/终端/CLI
     ▼
     S（黑色）
```

这创造了以下特点：
- **视觉清晰**：立即传达"代码导航指南"
- **象征意义**：指南针 = 找到方向；`>` = 代码/终端
- **可缩放性**：适用于从 16px 到 512px 的任何尺寸
- **品牌一致性**：以极简配色匹配开发者工具美学

## 色彩系统

### 调色板

| 颜色 | 十六进制 | RGB | 用途 |
|-------|-----|-----|-------|
| 黑色（主色） | `#000000` | 0, 0, 0 | 主要描边、文本、南向指针 |
| 白色（背景） | `#FFFFFF` | 255, 255, 255 | 浅色背景 |
| 灰色（辅助色） | `#6B7280` | 107, 114, 128 | 次要刻度标记、次要文本 |
| 亮绿色（强调色） | `#22C55E` | 34, 197, 94 | 北向指针、中心点、强调线 |
| 近黑色（深色背景） | `#0A0A0A` | 10, 10, 10 | 深色模式背景 |

### 对比度比率（WCAG）

- 白底黑字：**21:1** AAA级
- 白底灰字：**4.6:1** AA级
- 白底绿字：**3.2:1**（仅用于装饰，不用于文本）
- 深底白字：**19.5:1** AAA级

### 强调色规则

**亮绿色（#22C55E）仅用于高亮：**
- 指南针北向指针
- 中心点
- 强调下划线/边框
- 绝不作为背景色使用
- 绝不用作正文颜色

## 图标细节

### 指南针规格

指南针标记由以下几何元素构成：

```
元素         | 描边/填充      | 颜色
-------------|--------------|------------------------
外环         | 3px 描边     | 黑色 / 白色（深色模式）
北向刻度     | 2.5px 描边   | 黑色 / 白色（深色模式）
其他主方向刻度| 2px 描边     | 灰色 / 白色 50%（深色模式）
间向刻度     | 1.5px 描边   | 灰色 / 白色 40%（深色模式）
北向指针     | 填充多边形    | #22C55E（始终绿色）
南向指针     | 填充多边形    | 黑色 / 白色（深色模式）
> 括号       | 3px 描边     | 黑色 / 白色（深色模式）
中心点       | 填充圆形     | #22C55E（始终绿色）
```

### 尺寸递进

```
16px  → 仅保留圆环 + 指针 + 箭头（最小化）
32px  → 添加主方向刻度标记
64px  → 添加间向刻度标记
128px → 完整细节，所有元素清晰
256px → 最大细节，加粗描边
```

## 自定义

### 更改强调色

要创建不同强调色的变体：

1. 将所有 `#22C55E` 实例替换为你的强调色
2. 确保装饰元素的对比度比率保持在 3:1 以上
3. 保持黑/白/灰结构不变

### 缩放

```css
svg {
  width: 256px;
  height: 256px;
}
```

SVG 通过 viewBox 自动缩放——无需使用 transform。

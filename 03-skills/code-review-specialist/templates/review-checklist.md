# 代码审查清单

## 安全检查清单
- [ ] 无硬编码的凭据或密钥
- [ ] 对所有用户输入进行输入验证
- [ ] SQL 注入防护（参数化查询）
- [ ] 对状态变更操作进行 CSRF 防护
- [ ] 通过适当的转义防止 XSS 攻击
- [ ] 对受保护端点进行身份验证检查
- [ ] 对资源进行授权检查
- [ ] 安全的密码哈希（bcrypt、argon2）
- [ ] 日志中无敏感数据
- [ ] 强制启用 HTTPS

## 性能检查清单
- [ ] 无 N+1 查询
- [ ] 合理使用索引
- [ ] 在有益处的地方实现缓存
- [ ] 主线程上无阻塞操作
- [ ] 正确使用 async/await
- [ ] 大数据集分页处理
- [ ] 数据库连接池化
- [ ] 正则表达式已优化
- [ ] 无不必要的对象创建
- [ ] 防止内存泄漏

## 质量检查清单
- [ ] 函数 < 50 行
- [ ] 变量命名清晰
- [ ] 无重复代码
- [ ] 正确的错误处理
- [ ] 注释解释 WHY（为什么），而非 WHAT（是什么）
- [ ] 生产环境中无 console.log
- [ ] 类型检查（TypeScript/JSDoc）
- [ ] 遵循 SOLID 原则
- [ ] 正确应用设计模式
- [ ] 代码自文档化

## 测试检查清单
- [ ] 已编写单元测试
- [ ] 覆盖边界情况
- [ ] 测试了错误场景
- [ ] 集成测试到位
- [ ] 覆盖率 > 80%
- [ ] 无不稳定（flaky）测试
- [ ] Mock 外部依赖
- [ ] 测试命名清晰
